Itemsets - roster 1.7 addon - Last updated 21/05/2007

With this addon you can keep track of any itemsets part you're having or missing

Supported sets:
Quote::

Dungeon Set 1
Dungeon Set 2 (0.5)
Dungeon Set 3 (3.5)
Tier 1
Tier 2
Tier 3
Tier 4
ZG
AQ20
AQ40
PVP Rare
PVP Epic

Multi-language support:
Quote::
to enable it you have to change
$addon_conf['itemsets']['multilanguage'] = 0;
to
$addon_conf['itemsets']['multilanguage'] = 1;
in the conf.php
